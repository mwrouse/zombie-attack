# Zombie Attack!
This is my first HTML/Javascript game.

It is a recreation of a game that I created while in high school in QBasic.



## Controls
The controls for the game are simple:

ESC - Pause

W/&uarr; - Move upwards

A/&larr; - Move left

S/&darr; - Move downwards

D/&rarr; - Move right

SPACE - Attack
